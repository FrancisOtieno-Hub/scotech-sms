Setup Guides

üì± Africa's Talking (Recommended for Kenya)

  1. Visit africastalking.com and create an account
  2. Verify your account and complete the registration
  3. Go to your dashboard and navigate to Settings ‚Üí API Key
  4. Copy your Username and generate an API Key
  5. (Optional) Apply for a Sender ID - this makes SMS appear from your school name
  6. Enter your credentials above and click "Save Configuration"
  7. Click "Send Test SMS" to verify everything works
üí∞ Pricing: Approximately KES 0.80 per SMS in Kenya

üì± Twilio (International)

  1. Visit twilio.com and sign up
  2. Complete the verification process
  3. Purchase a phone number from Twilio's console
  4. Go to Console ‚Üí Account ‚Üí Keys & Credentials
  5. Copy your Account SID and Auth Token
  6. Enter your credentials and phone number above
  7. Click "Save Configuration" and test
üí∞ Pricing: Approximately $0.07 per SMS (varies by country)

üîí Security Notice

Your API credentials are sensitive and should be kept secure. In production:

  • API keys are encrypted before being stored in the database
  • Only school administrators can view or modify SMS configuration
  • Never share your API credentials with unauthorized users
  • Regularly monitor your SMS usage for any unusual activity
  • If you suspect your credentials have been compromised, regenerate them immediately
